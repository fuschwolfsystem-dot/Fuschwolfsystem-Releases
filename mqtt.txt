📷 AI Vision Subsystem
These topics handle the high-speed data passing between the Python AI core and the C++ HMI.

Topic: fuschwolf/vision/stream

Payload: Base64 Encoded String (JPG image bytes)

Action: Streams the live, compressed video frames directly into the dashboard UI.

Topic: fuschwolf/vision/detection

Payload: JSON Object (e.g., {"class": "BIG MINERAL WATER", "confidence": 98.5, "height": 320, "status": "LOGGED"})

Action: Triggers the UI to log a new entry into the Product Quality table and prints detection metrics to the console.

📊 Admin Efficiency Dashboard
These topics drive the live animations for the gauges and charts on Desktop 7. The C++ engine strictly parses these as numeric floats.

Topic: fuschwolf/admin/oee

Payload: Float (0.0 to 100.0)

Action: Updates the "Overall Equipment Effectiveness" dial gauge.

Topic: fuschwolf/admin/ai

Payload: Float (0.0 to 100.0)

Action: Updates the "AI Defect Catch Rate" dial gauge.

Topic: fuschwolf/admin/uptime

Payload: Float (0.0 to 100.0)

Action: Updates the "System Uptime" dial gauge.

Topic: fuschwolf/admin/throughput

Payload: Float (Numeric Volume)

Action: Plots a new X/Y point on the "Production Throughput" live scrolling line chart.
